Installing the calendar:

1) as you can see there are 5 files in the zip file

  • calendar.swf - this is the actual flash engine
  • calendar-install.php - this is the calendar installation file
  • calendar-admin.php - php administration file. Using this web page you can manage your web calendar data and settings.
  • calendar-data.php - php script to read data from MySQL database and send it to the flash engine
  • help.html - the file you are reading now :)

    • 2) before running the installation script you will need to edit the 3 PHP files and set the correct login details. You can edit the files using a plain text editing software such as notepad. At the top of the 3 PHP files you will see:

    $hostname = 'localhost';
    $mysql_user='';
    $mysql_pass='';
    $mysql_database='';


  • $hostname = 'localhost'; - this is your MySQL server. Most probably localhost will work but it may also be IP address or valid domain name.
  • $mysql_user=''; - this is the username that you use to connect to your mysql database.
  • $mysql_pass=''; - this is the password that you use to connect to your mysql database.
  • $mysql_database=''; - this is the database name where the 2 data tables will be created.

    • In case you do not know some of the above data, please contact your hosting provider. Please, note that the user that you will use for the web calendar should have INSERT, SELECT, UPDATE, DELETE, CREATE TABLE permissions.

    Below these few lines in the calendar-admin.php file you will see the default login details for your web calendar administration page. You can change it to whatever you want.

    $admin_username = 'admin';
    $admin_password = 'pass';

    3) upload the 3 PHP files and the calendar.swf file in a folder on your web server

    4) open http://www.yoursite.com/folder/calendar-install.php, where yoursite.com is the domain name and 'folder' is the folder where you uploaded the files. You will be asked to confirm the login details for your MySQL server. On the next step you will have 2 new database tables for your calendar.

    5) delete the calendar-install.php file

    6) all you have to do now is to login at http://www.yoursite.com/folder/calendar-admin.php and manage your calendar via the administration page
